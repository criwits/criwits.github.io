一台新 Windows 电脑的折腾日记

2020年8月4日
| 技术
| 阅读

近日，Hans 的一朋友 Green Dragonfly 购买了一台新机，型号是联想的 Y7000P（i7-10875H，32GB 内存，1TB 固态硬盘，预装 Windows 10）。入手第二天，他便携机来访寒舍，想一起给电脑安些常用软件，做些优化。为使得我们的折腾过程有个记录，特开此文，一一记之。

软件篇

网络软件

浏览器：Firefox

下载地址：Mozilla 官网

我喜欢 Firefox 浏览器。这似乎有些另类——毕竟现在浏览器市场可以算是 Chrome 的天下。但 Firefox 依然有着她迷人的一面：自由与开放。Mozilla 基金会给我的印象非常好，好在它推出了一系列尊重用户隐私权的好产品，更好在它维护着 Firefox 这一庞大的开源项目。Green Dragonfly 似乎也很认同这一点，在达成共识以后，我们决定使用 Firefox 作为默认浏览器。

然而，下载 Firefox 时需要注意，由“北京谋智火狐”开发的“火狐浏览器”是万万不可碰的。这个所谓的中国版 Firefox 可能有着我们无法发现的后门，以及“中国特色”的广告。因此，下载 Firefox 时，一定认准 Mozilla 基金会的网站。

不可否认，在 Chrome 的天下，我们仍然需要一份 Chromium 内核的浏览器以备不时之需。Chromium 内核的新版 Edge 可能是一个不错的选择。但这并不影响 Firefox 的默认浏览器地位。Chromium 浏览器这时是一种“我可以不用，但不能没有”的存在。

建议使用 Bing 作为默认搜索引擎。不想用百度，就这样。

喜欢 Firefox 很大程度上是由于插件。这里自然也推荐一批优秀的 Firefox 插件。（直接在 Mozilla 的附加组件中心搜索即可）

  • ADBlocker Ultimate：屏蔽大部分广告、防止跟踪。
  • Aria2 Download Manager Integration：用于和 Motrix 联动，提升下载体验。后文会提及 Motrix。

下载工具：Motrix

下载地址：Motrix

Motrix 是一款简约、干净而又不凡的下载工具。它采用 aria2 作为下载内核，能够通过插件与 Firefox、Chrome 等多种浏览器联动，且支持 BT 种子下载、支持 thunder:// 迅雷链接下载，体验极佳。

若要将 Motrix 与 Firefox 联动，需要安装上文提到的 Aria2 Download Manager Integration 插件，然后将这个插件的控制页面下“RPC 服务器”选项页中“端口号”从 6800 设置为 16800，随后保存退出。这时，在 Firefox 中所有的下载，都会被发送到 Motrix 中，实现无缝联动。（Motrix 需要常驻后台，建议给它设置开机启动。）

工具软件

搜索文件软件：Everything

下载地址：Everything

基于一种神奇的索引技术，Everything 能实现极其快速的文件搜索。

安装过程中，所有选项保持默认即可。

设置搜索快捷键（例如：Alt-F）的方法：工具——选项——键盘——显示窗口热键：设置成 Alt-F。

远程协助软件：Anydesk

下载地址：Anydesk

Anydesk 是一个跨平台（含Windows、macOS、Linux 以及移动平台 Android 和 iOS）的远程协助软件。小巧、轻量是其特点之一。毕竟在今天，只有几 MB 的软件可谓少见。

值得注意的是，Anydesk 好虽好，但会开机启动，且应用内没有关闭开机启动的设置。其开机自启的策略十分“原始”：将自己的快捷方式添加到“启动”文件夹。众所周知，“启动”文件夹下的程序，会在 Windows 登陆后自动启动。所以我们只需要删除 C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp 和 C:\Users\用户名\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup 下的“Anydesk”快捷方式（如果有）即可。

压缩软件：7-zip

下载地址：7-zip

小巧、轻量、性能强大，这是我要的 7-zip！

安装完成后，可以打开“7-zip 文件管理器”，在选项中设置好文件关联。

通讯软件

聊天 & 语音通话软件：Skype

下载地址：UWP 版本 Windows 10 自带。

参见我年初的文章：

　　说起 Skype 了。Skype 确实还是很优秀的。真正使用 Skype 大概是 16 年暑假那会，在 Skype 上加了一个 Linux 技术交流群，才真正认识了这款软件。它优秀的地方主要有：

  • 在国内可正常使用（无需特殊网络环境）
  • 有已读回执（这个真的很重要，尤其是当你和别人聊天需要控制聊天节奏的时候）
  • 极为优秀的语音通话质量（然而没亲自体验）
  • 界面好看
  • 具有端到端加密通信功能 　　缺点也有：
  • 在国产手机上不推送通知
  • 国内发图片速度极慢
  • 能聊天的朋友太少。

事实上，通话质量在今天体验之后，确实称得上“极为优秀”。而“在国产手机上不推送通知”这点，可以折中使用 Skype Lite 这个“印度专供版”来解决。至于“国内发图片速度极慢”，则很大程度上是网络问题。

顺便记一下，Linux 版本的 Skype 在这里下载，Android 版可以在这里选择下载。与 Firefox 类似，请注意不要下载到光明网代理的“中国版 Skype”。

通讯软件：QQ

下载地址：Microsoft Store，搜索“QQ 桌面版”。

在 Microsoft Store 中上架的这个“QQ 桌面版”，其实就是普通的 win32 版本的 QQ。不同之处在于广告少了，进程占用少了，所以还是推荐安装的。毕竟你活在今天不能不用 QQ 吧。

影音软件

播放器：PotPlayer

下载地址：PotPlayer

由于国际版官网被墙，只能走这个“PotPlayerCN”的国内站点下载。不过这似乎是个个人站点，应该没有什么后门。如果有同学找到了更好的下载 PotPlayer 的地方，欢迎联系我。

录屏 & 直播：OBS Studio

下载地址：OBS Studio

OBS Studio 算是直播界享有盛名的软件了吧。非常适合各类直播、录屏使用，并且好像还可以利用 NVIDIA 显卡的某些技术来提升性能。

关于 OBS Studio 的配置，可以参考 bilibili 上的这个视频。

视频制作：DaVinci Resolve

下载地址：DaVinci Resolve

DaVinci Resolve（达芬奇）也是影视后期界的重要软件之一，其卓越的调色性能被广泛地用于电视、电影等节目的后期制作中。事实上，除了调色，达芬奇本身也是一个完整的非线性视频编辑软件。更重要的是，相比起 Adobe Premiere Pro 这种不仅收费贵而且还是订阅制的软件，达芬奇的基本功能是免费的。能用正版为啥还要用盗版呢？

bilibili 上的著名 up 主“影视飓风”制作了一套达芬奇的基础教程，可以与达芬奇软件配套观看使用。

一个小小的总结

可以看到，我们选择软件大多以小巧、轻量、免费、开源等为选择标准。小巧轻量，意味着在存储空间有限的情况下，能让我们有更多的余地放下更多应该放下的东西。免费，意味着我们不需要跑遍全网冒着下到病毒的奉献寻找一个“破解版”。开源，意味着我们的隐私安全能够得到保障。并不是说庞大的、收费的、闭源的软件都不好，只是在互联网的洪流面前，小巧轻量、免费开源能够让我们把电脑用的更顺手罢了。

如果大家还有什么优秀软件和实用技巧，欢迎联系 Hans。我的联系方式在网站册侧边栏里有哦。
